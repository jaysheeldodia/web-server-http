# About This Documentation

A comprehensive documentation website for the high-performance C++ WebServer project. This documentation provides detailed guides, API references, and examples for developers.

## Live Documentation

The documentation is built with MkDocs. Build with `mkdocs build`; preview with `mkdocs serve`.

## Documentation Structure

- **Getting Started** – Overview, Quick Start, Installation, Configuration
- **Architecture** – System design, Thread Pool, HTTP/2, WebSocket
- **API Reference** – REST endpoints, WebSocket API
- **Admin Dashboard** – Real-time monitoring
- **Development** – Building, Testing, Benchmarks

## Local Development

To preview the documentation locally:

```bash
mkdocs serve
# Open http://127.0.0.1:8000
```

To build static HTML:

```bash
mkdocs build
# Output in site/
```
